# EMI Calculator

A simple EMI Calculator built using Spring Boot.

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- HTML
- CSS

## Features
- Calculate monthly EMI
- User-friendly interface
- Responsive design

## Project Flow
1. User enters Principal Amount
2. User enters Interest Rate
3. User enters Time (Years)
4. System calculates EMI
5. Result displayed on screen
